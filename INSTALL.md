# Quick Start Guide

## Installation Steps

1. Open Chrome and go to: `chrome://extensions/`

2. Enable **Developer mode** (toggle in top-right)

3. Click **"Load unpacked"**

4. Select the `pokellector_downloader` folder

5. Done! The extension is now installed.

## Test It Out

1. Visit: https://www.pokellector.com/EX-FireRed-LeafGreen-Expansion/Charmander-Card-57

## Alternative: Manual Injection

If the button doesn't appear automatically:
1. Click the extension icon in your toolbar
2. Click "Inject Download Button"

---

**Note**: The icons are SVG files saved as `.png` extensions. Chrome will still load them, but for production use, you may want to replace them with actual PNG/JPG icons using an image editor.
